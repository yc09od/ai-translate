# 隐私政策

**最后更新日期：2026年3月17日**

感谢您使用 **[ai-translate]**（以下简称"本服务"或"我们"）。本隐私政策说明我们如何收集、使用、存储和保护您的个人信息。请在使用本服务前仔细阅读本政策。

---

## 1. 信息收集

### 1.1 账户信息（通过 OAuth 登录）

本服务不支持密码注册。您必须通过以下 OAuth 提供商登录：

- **Google（Gmail）**
- **Microsoft（Hotmail/Outlook）**

登录时，我们从 OAuth 提供商获取并存储以下信息：

| 数据项 | 来源 | 用途 |
|--------|------|------|
| 电子邮件地址 | OAuth 提供商 | 账户唯一标识 |
| 用户名（显示名称） | OAuth 提供商 / 用户自定义 | 界面展示 |
| OAuth 提供商类型 | OAuth 提供商 | 区分登录来源 |

我们**不会**获取您的 OAuth 密码、联系人列表、日历、邮件内容或其他与本服务无关的数据。

### 1.2 语音数据

当您使用实时翻译功能时：

- 您的麦克风音频将实时采集并通过加密连接（WebSocket）传输至服务器。
- 音频数据将发送至第三方 AI API（Google Gemini API 或 Moonshot Kimi API）进行语音识别与翻译处理。
- 处理完成的音频文件临时保存于服务器，用于调试和质量分析，**不用于训练 AI 模型**。
- 我们建议您不要在翻译中包含敏感个人信息（如身份证号、银行卡号、医疗信息等）。

### 1.3 翻译内容

- 语音识别结果（原文）和翻译结果（译文）将存储至我们的数据库（MongoDB），与您的账户及所属主题关联。
- 翻译记录用于提供"翻译历史"功能，供您查阅和导出。

### 1.4 主题（Topic）数据

- 您创建的主题名称及排序信息存储于我们的数据库。

### 1.5 会话与令牌

- 登录成功后，服务器会签发 **JWT（访问令牌）** 和 **刷新令牌**，并存储于服务器端 Redis 缓存（含有效期限制）。
- 令牌用于身份验证，不包含您的密码或其他敏感信息。
- 您注销后，服务器将立即删除对应的会话令牌。

### 1.6 用户偏好

- 您选择的界面语言（中文 / English）存储于您的本地浏览器（localStorage），不上传至服务器。

## 2. 信息使用

我们收集的信息仅用于以下目的：

1. **提供核心功能**：实时语音翻译、历史记录管理、主题管理。
2. **身份验证**：识别您的账户，保障服务安全。
3. **服务改进**：分析使用情况以优化翻译质量和用户体验（匿名汇总数据）。
4. **技术支持**：诊断错误和技术问题。

我们**不会**：
- 将您的个人信息出售给第三方。
- 将您的翻译内容用于广告定向。
- 在未获明确同意的情况下分享可识别您身份的信息。

## 3. 信息共享

### 3.1 第三方 AI 服务

为提供翻译功能，您的语音/文本数据将发送至以下第三方服务：

- **Google Gemini API**（由 Google LLC 提供）：适用 [Google 隐私政策](https://policies.google.com/privacy)
- **Moonshot Kimi API**（由 月之暗面（北京）科技有限公司提供）：适用 Moonshot AI 隐私政策

请注意，这些第三方服务有各自独立的隐私政策，我们建议您了解其数据处理方式。

### 3.2 OAuth 提供商

您通过 Google 或 Microsoft 登录时，相关认证流程由其提供商处理，适用其各自的隐私政策：

- Google：[https://policies.google.com/privacy](https://policies.google.com/privacy)
- Microsoft：[https://privacy.microsoft.com](https://privacy.microsoft.com)

### 3.3 法律要求

在法律要求的情况下（如法院命令、政府监管），我们可能依法披露相关信息。

## 4. 数据存储与安全

- **存储位置**：用户数据存储于我们托管的 MongoDB 数据库，会话数据存储于 Redis 缓存。
- **传输安全**：所有客户端与服务器之间的通信均通过 HTTPS / WSS 加密传输。
- **访问控制**：所有 API 端点均需有效 JWT 令牌才能访问；翻译记录仅对记录所有者可见。
- **数据隔离**：每位用户只能访问自己的主题和翻译历史，系统在接口层强制执行归属校验。

尽管我们采取合理的安全措施，但任何网络传输和存储系统均无法保证绝对安全。

## 5. 数据保留

| 数据类型 | 保留期限 |
|----------|----------|
| 账户信息 | 账户存续期间，注销后按请求删除 |
| 翻译记录 | 用户主动删除主题或账户注销前持续保留 |
| 会话令牌 | 访问令牌：7天；注销时立即删除 |
| 音频文件 | 临时存储，定期清理（不超过30天） |
